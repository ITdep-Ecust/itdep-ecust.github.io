## 华东理工大学勤工助学IT部技术文档

## 前言

把文档保存在何处、如何有序地编排、怎样方便同事们调取，一直是IT部技术文档保存工作的痛点。

技术文档相当重要，曾经知识库网站丢失，为IT部的基础工作造成了很大的麻烦。只有把地基打牢，IT部才有可能不断挑战新的高度。

使用Github Pages，是如今文档站的一个主流，包括大学课程共享、使用教程等等。

> “如果我看得比别人更远些，那是因为我站在巨人的肩膀上。”

> “我为这一年一年，这么多人孤军奋战的重复劳动感到不平。我希望能够将这些隐晦的、不确定的、口口相传的资料和经验，变为公开的、易于获取的和大家能够共同完善、积累的共享资料。我希望只要是前人走过的弯路，后人就不必再走。这是我的信念，也是我建立这个项目的原因。”
